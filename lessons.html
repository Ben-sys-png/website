HTML Basics
  • !Doctype - Every HTML document needs a !DOCYTPE html.
  • <h1> - This is biggest heading tag, for page titles.
  • <p> - A paragraph of text.
  • <a> - A link to another page or website.
  • <img> - Adds an image to the page.
  • <ul> - A bullet point list.
  • <ol> - A numbered list.
  • <table> - Shows information in rows and columns.
  • <strong> - Defines text with strong importance by displaying content in bold.

HTML Example

This example uses the HTML tags above:

    <!DOCTYPE html>
    <html>
    <head>
    <title>HTML Example</title>
    </head>
    <body>

    <h1>My Web Page</h1>
    <p>This is a paragraph.</p>
      <a href="https://exampleLink.com">Visit Example</a>
      <img src="images/pic.jpg" alt="HTML & CSS code example">

      <ul>
        <li>Item 1</li>
        <li>Item 2</li>
      </ul>

      <ol>
        <li>First</li>
        <li>Second</li>
      </ol>

      <table>
        <tr><th>Name</th><th>Age</th></tr>
        <tr><td>Alice</td><td>25</td></tr>
      </table>

      </body>
      </html>

CSS Basics
  • color - Changes text color.
  • background-color - Changes background color.
  • font-size - Makes text bigger or smaller.
  • margin - Space outside an element.
  • padding - Space inside an element, around the text.
  • border - Adds a line around an element.

Different CSS box displays:

  • display: none - Hides an element.
  • display: inline - Displays an element inline with other elements. Any height and width properties will have no effect.
  • display: block - Takes up full available width of parent container, unless a specific width is defined and starts on a new line.
  • display: inline-block - Like inline elements,but allows other elements to sit alongside them on the same line.

CSS Example

This example demonstrates some basic CSS properties:

    /* Changes text color */
    p {
      color: #003366;
    }

    /* Changes background color */
    div {
      background-color: #f0f0f0;
    }

    /* Makes text bigger or smaller */
    h1 {
      font-size: 24px;
    }

    /* Space outside an element */
    section {
      margin: 20px;
    }

    /* Space inside an element, around the text */
    button {
      padding: 10px;
    }

    /* Adds a line around an element */
    img {
      border: 2px solid black;
    }
Accessibility
  • Always add alt text to images.
  • Make text easy to see with good contrast.
  • Use headings in the right order.
  • Make sure the site works without a mouse.
  • Dont use only color to show meaning.
  • Use clear, descriptive link text.
